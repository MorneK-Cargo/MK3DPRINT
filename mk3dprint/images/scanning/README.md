# 3D Scanning Application Images

This directory contains images showcasing the three main scanning applications.

## Required Images

### 1. auto-parts.jpg
**Purpose:** Showcase auto parts and reverse engineering capabilities
**Content:** Automotive components, engine parts, door panels, brackets, mechanical parts
**Recommended Specs:**
- Dimensions: 800x600px minimum (aspect ratio: 4:3)
- Format: JPG, optimized
- File size: ~100-150KB
- Description: Clear shots of scanned automotive/mechanical components

### 2. medical.jpg
**Purpose:** Showcase medical and dental scanning capabilities
**Content:** Dental models, foot/ankle models, orthotic devices, medical reproductions
**Recommended Specs:**
- Dimensions: 800x600px minimum (aspect ratio: 4:3)
- Format: JPG, optimized
- File size: ~100-150KB
- Description: Clear shots of scanned medical and dental models

### 3. reverse-engineering.jpg
**Purpose:** Showcase reverse engineering and CAD model creation
**Content:** Complex parts, detailed components, precision items ready for CAD conversion
**Recommended Specs:**
- Dimensions: 800x600px minimum (aspect ratio: 4:3)
- Format: JPG, optimized
- File size: ~100-150KB
- Description: Clear shots demonstrating reverse engineering capability

## Notes
- All images display at 240px height in the card layout
- Images use `object-fit: cover` so they'll maintain aspect ratio
- Hover effect includes slight zoom (1.05x scale)
- Images should be crisp and well-lit to showcase scanning quality
